# Project Daybreak - YJCC Events Platform

מערכת ניהול אירועים מודרנית עבור YJCC, מאפשרת ניהול משתתפים, רישום והודעות WhatsApp אוטומטיות.

## תכונות עיקריות
- ניהול משתתפים ורישום לאירועים
- שליחת הודעות WhatsApp אוטומטיות
- ממשק משתמש מודרני ונוח
- תמיכה מלאה בעברית (RTL)
- עיצוב קהילתי וחם

## התקנה והרצה

### דרישות מקדימות
- Node.js v18 ומעלה
- npm v9 ומעלה

### התקנה
```bash
# התקנת תלויות
npm install

# הרצה בסביבת פיתוח
npm start

# בנייה לייצור
npm run build

# העלאה לאוויר
npm run deploy
```

## טכנולוגיות
- React
- Material-UI
- Emotion (CSS-in-JS)
- WhatsApp API
- Netlify (hosting)

## מבנה הפרויקט
```
src/
  ├── components/     # רכיבי React
  ├── context/       # Context API
  ├── hooks/         # Custom Hooks
  ├── styles/        # עיצוב וסגנונות
  ├── utils/         # פונקציות עזר
  └── assets/        # תמונות וקבצים
```
